Exercice : Les données personnelles

Cochez, d'après-vous, les données que l'on nomment données personnelles

Votre choixChoix attenduRéponse

  • Votre nom,

  • Votre age,

  • L'Adresse IP de ton ordinateur,

  • L'adresse IP de l'imprimante de votre école,

  • La localisation du smartphone de votre père.

Une donnée est un élément se rapportant à un objet, une personne ou un événement.

Une donnée personnelle est une donnée identifiant directement ou indirectement une personne physique.
